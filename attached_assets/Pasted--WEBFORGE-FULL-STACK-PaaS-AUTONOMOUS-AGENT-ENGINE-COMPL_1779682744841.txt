1. User Command Deck: A fluid, conversational ChatGPT-style Telegram Bot interface.
2. Administrative Controller: A secondary Telegram Bot that processes manual OPay bank transfer receipts via inline approval keys.
3. Brain Layer (Intelligent Router): An abstraction layer that dynamically routes planning, coding, and fixing tasks to specialized models via an external OpenAI-compatible gateway.
4. Open-Source Muscle Layer: Local integrations of Ruflo (State/Concurrency) and OpenClaw (File CRUD/Terminal).
5. Isolated Runtimes: Sandbox project directories that run separate background node apps or custom user bots.
6. Real-Time Workspace: A split-pane web UI streaming file dabs via SSE and rendering app states via a dynamic port proxy.
