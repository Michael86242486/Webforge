# Coca-Cola Advertiser Application

The Coca-Cola Advertiser Application is a simple web application built using Node.js and Express that allows users to log in and access Coca-Cola advertisements.

## Description
This application is designed to demonstrate a basic login system and advertisement display. The application features a user login page and a main page that displays Coca-Cola advertisements. The application is built using Node.js and Express for the backend and vanilla JavaScript for the frontend.

## Tech Stack
* Node.js
* Express
* HTML
* CSS
* JavaScript

## Features
The following files are included in this project:

* `package.json` — Project manifest
* `src/index.js` — Express server
* `src/routes/api.js` — API routes
* `public/index.html` — Main HTML
* `public/style.css` — Styles
* `public/app.js` — Frontend JS
* `README.md` — Documentation
